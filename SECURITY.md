# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in PhantomLink, please **do not** open a public issue.

Instead, email the details to the maintainer.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅ Active |

## Security Architecture

- PhantomLink operates exclusively against known scam platforms
- No real WhatsApp accounts are used or compromised
- All operational logs are encrypted with AES-256-GCM
- Device-locked execution via hardware dead switch
- Tor routing enabled by default
